<h2>父亲的派克钢笔</h2>
<p>这是父亲留给我的派克钢笔，笔尖是 14K 金的。父亲在纺织厂做了大半辈子账房先生，用这支笔记了几十年账，每一笔都工工整整。他说，做账如做人，一笔一划都得对得起良心。</p>
<p>现在这钢笔已经写不出水了，笔杆上的黑漆也磨掉了一大半，露出里面的黄铜。但每次看到它，我就想起父亲伏在案头的身影——台灯昏黄，算盘声脆，钢笔沙沙。老人家要是知道我现在还在用笔——虽然换成了圆珠笔——写着这些字，大概会笑着点点头吧。</p>
<p>有时候我想，一个人一生最珍贵的遗产，不是房子不是存款，是这样一支笔。它里面有父亲的手温，有他对"认真"二字的坚守。</p>

<h2>结婚时的粉彩茶具</h2>
<p>一套粉彩茶具，一壶四杯，是我和老陈结婚时他母亲送的。据说是景德镇的老物件，白瓷胎上画着牡丹——红的、粉的、紫的，在莹白的釉面上开得热闹。六十年了，杯子打碎了一个，壶盖的钮也掉了，但剩下的我们都还留着。</p>
<p>每年过年，我都会用这把壶泡一壶龙井。老陈走了以后，我一个人对着这壶茶，好像他还在旁边坐着——话不多，就那么静静地陪着。茶凉了，也没有人再来续了。</p>
<p>这套茶具见证了一个家的开始。它不完整了，但还在。有些东西坏了仍旧是宝贝。</p>

<h2>退休后的手抄诗集</h2>
<p>退休那天，老陈送了我一本笔记本。从那以后，一本接一本，我用毛笔和钢笔写了十几本。里面有诗——多半是打油诗；有生活随笔，记录院子里的桂花开了、隔壁的猫生了小猫、孙女第一次叫奶奶；有给老陈和孩子们的信——有些寄出去了，有些没寄；还有一些抄来的喜欢的句子，鲁迅的、张爱玲的、汪曾祺的。</p>
<p>字迹越来越潦草，因为手越来越不听话了。但每次翻开，就像走进了另一个时代。第一本的字还工整有力，第十本已经开始歪歪扭扭——这就是衰老的痕迹吧。但我不怕，因为这些本子里装着的东西，比年轻更重要。</p>
<p>孙女说，这些本子应该做成一本书。我说不急，等我再写几本。人活到八十多岁，最珍贵的就是这些日积月累的东西——不是什么大道理，是每一天里真实的欢喜和忧伤。</p>
