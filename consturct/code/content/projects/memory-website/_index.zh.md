## 我长大的那条弄堂

虹口的弄堂，现在大多拆了。但闭上眼睛，我还闻得到那里的气味——清晨煤炉的烟火味，夏天井水里冰镇的西瓜，隔壁阿婆晒的梅干菜。弄堂不长，从头走到尾只要五分钟，但那五分钟里装下了一整个世界。

我们住在二楼，木楼梯踩上去嘎吱响。窗户对着天井，晾衣杆上永远挂着各家各色的衣服。到了傍晚，收音机里传出评弹的声音，大人们搬出竹椅在弄堂口乘凉，小孩子满世界跑。那时候日子不富裕，但邻里之间像一家人。谁家包了馄饨，总要端一碗给隔壁。谁家孩子没人看，总有阿婆帮忙照应。

现在的年轻人住进了高楼，关上门谁都不认识谁。我有时候想，舒适是舒适了，但有些东西，弄堂没了也就没了。

## 海峡那头的信

1962 年，我收到了一封从香港转寄来的信。信封已经皱得不成样子，邮票贴了三张。是我中学最好的朋友——林淑珍写来的。她四九年跟家人去了台湾，从此断了音讯。

那封信我读了不知多少遍。她说她在台北也做了老师，教国文。她说台北的冬天也冷，但不像上海那样湿。她说她很想念我，想再跟我一起去城隍庙吃小笼馒头。

后来，我们断断续续通了三十多年的信。信里不谈政治，只谈书、谈花、谈学生、谈儿女。九十年代，她终于回了一趟上海。我们在外滩见的面，两个老太太抱在一起哭。她头发白了，牙齿也缺了几颗，但笑起来还是十六岁的样子。

这个故事我讲给孙女听，她说奶奶你们太不容易了。我说，不容易是常态，但守住那些珍贵的人和事，就是生活的意义。

![旧信件与老照片](https://picsum.photos/seed/old-letters/1200/600)

## 教了三代人

教学生涯里最奇妙的事——我教过一家三代人。

先是七十年代教过一个男孩周建国，他聪明但调皮，我让他当了语文课代表，这孩子一下子就上进了。后来他考上了大学，做了工程师。九十年代，他的女儿周小燕又分到了我班上。我一看名字和地址，就问："你爸爸是不是叫周建国？"小姑娘瞪大了眼睛。再后来，二〇一几年，周小燕带着她的儿子来看我，说想让儿子也叫我一声陈老师。

我说，别叫陈老师了，叫我奶奶吧。

这样的故事，当老师的人一辈子总能遇到几个。它们让我觉得，站在讲台上的那四十年，值了。

## 父亲的钢笔

这是父亲留给我的派克钢笔，笔尖是 14K 金的。父亲用它记了几十年账，每一笔都工工整整。他说，做账如做人，一笔一划都得对得起良心。

现在这钢笔已经写不出水了，笔杆上的黑漆也磨掉了一大半，露出里面的黄铜。但每次看到它，我就想起父亲伏在案头的身影。老人家要是知道我现在还在用笔——虽然换成了圆珠笔——写着这些字，大概会笑着点点头吧。

![老派克钢笔](https://picsum.photos/seed/fountain-pen-vintage/800/600)

## 结婚时的瓷茶具

一套粉彩茶具，一壶四杯，是我和老陈结婚时他母亲送的。据说是景德镇的老物件，上面的牡丹画得极好。六十年了，杯子打碎了一个，壶盖的钮也掉了，但剩下的我们都还留着。

每年过年，我都会用这把壶泡一壶龙井。老陈走了以后，我一个人对着这壶茶，好像他还在旁边坐着。

![粉彩瓷茶具](https://picsum.photos/seed/chinese-tea-set/800/600)

## 退休后的手抄诗集

从 1996 年开始，我用毛笔和钢笔写了十几本。里面有诗——多半是打油诗，有生活随笔，有给老陈和孩子们的信，有一些抄来的喜欢的句子。字迹越来越潦草，因为手越来越不听话了。但每次翻开，就像走进了另一个时代。

孙女说，这些本子应该做成一本书。我说不急，等我再写几本。

![手抄笔记本](https://picsum.photos/seed/handwritten-notebook/800/600)

事实上，这个网站就是我想要的那本书——只不过比书更好，它可以让更多人看到，可以加照片，可以随时更新。父亲要是还在，大约会感叹时代的奇妙吧。
